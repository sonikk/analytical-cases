# Учебные кейсы по анализу данных

## Кейс 1  
Описание:  
• Таргетировать рекламу: найти категорию клиентов,
в которой платформа получает максимум прибыли
в расчёте на клиента.

## Кейс 2  
Задача:  
Отдел A создал продукт A: специальное
предложение, которое даёт доступ к контенту
при активности пользователя. Отдел B создал
аналогичный продукт B.  
Директор решил запустить только один продукт.
Руководители обоих отделов считают, что выбрать
должны именно их продукт.
